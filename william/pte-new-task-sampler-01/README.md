# William — PTE Task Practice

Hosted homework page for one Listening Fill in the Blanks (Type In) item, three fresh Write From Dictation items, one Reorder Paragraph item and one Write Essay item.

The page is an original PTE-style practice artifact, not an official Pearson test item. It preserves progress only in that browser's local storage and exports one text file for teacher return. Publishing the static page does not upload student answers to cloud storage.

Teacher reset route: append `#reset=1` to clear only this homework's saved state on the current browser and device.
